# 📦 TehBotolxESSE Library

Library utility untuk membantu developer FiveM dalam mengelola resource, khususnya untuk **version checking**, compatibility, dan modular helper lainnya.

Repository ini dibuat untuk mempermudah maintenance script agar selalu up-to-date dan terintegrasi dengan sistem distribusi (Tebex).

---

## 🚀 Features

- 🔄 **Auto Version Checker**
  - Mengecek versi script secara otomatis
  - Notifikasi jika ada update terbaru

- ⚙️ **Framework Support**
  - Compatible dengan ESX / QBCore / Standalone (customizable)

- 📦 **Reusable Library**
  - Bisa digunakan di berbagai resource
  - Modular & ringan

- 🔐 **Production Ready**
  - Cocok untuk script free maupun paid (Tebex)
